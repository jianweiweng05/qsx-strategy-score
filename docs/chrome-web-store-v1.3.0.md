## What's new

Version 1.3.0 completes multilingual result and artifact delivery for English, Chinese, Japanese, Korean, Spanish, and Brazilian Portuguese. The selected language now remains consistent across the score headline, grade, edge label, native/social sharing, PNG scorecard, three-page PDF, and optional email attachments.

The PDF is still generated by the official hosted scoring service from the same public core as GitHub. Core v0.3.3 wraps CJK text using measured font width, keeps punctuation on the correct line, embeds Japanese and Korean fonts reliably, and localizes all fixed PDF headings.

## Single purpose

QSX Strategy Score lets a user upload a tabular strategy backtest and receive a free, overfitting-aware strategy score, benchmark comparison, and diagnostic result. Version 1.3.0 corrects localization and downloaded-document presentation without adding a new purpose.

## Permission justification

- `storage`: saves the user's API base and language preference locally.
- `https://www.quantscopex.com/*`: calls the official scoring, scorecard, PDF, and optional email-delivery endpoints.
- `https://www.tradingview.com/*`: supports the existing TradingView page integration and free indicator link.

No new permission was added in 1.3.0. The extension does not execute remote code.

## Data handling disclosure

The strategy file and optional benchmark are sent only after an explicit scoring, share, download, or email action. Raw uploads are parsed in memory and are not retained by the scoring service. An email address is sent only when the user requests email delivery. It is added to the product-update contact list only when the separate marketing checkbox is selected.

Privacy policy: `https://www.quantscopex.com/privacy/chrome-extension`

## Reviewer steps

1. Open the extension popup and choose a language.
2. Upload a CSV return series, equity curve, or closed-trade log.
3. Click the scoring action and verify the localized headline, grade, edge, and next step.
4. Download PNG and PDF artifacts and verify that both use the selected language.
5. For Chinese, verify that long findings wrap inside the PDF panels.
6. Native sharing and X/Reddit titles should use the selected language.
7. Email delivery is optional and sends the localized PNG and PDF generated from the same uploaded file.
